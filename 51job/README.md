# 前程无忧招聘信息爬取 + 数据可视化

一共爬取了 运营 相关一共 99370 条职位信息，存为本地excel文件。

- 爬取时间：2021-03-03

- 实现目标：根据用户输入的关键字爬取相关职位信息存入本地，读取数据进行可视化展示（还没改。

- 涉及知识：请求库 requests、Xpath 语法、数据处理 Numpy、Pandas、数据可视化 Matplotlib。
